# Review Fetcher

A Svelte project for fetching and displaying reviews.

## Setup

1. Install dependencies:
   ```sh
   bun install
   ```
2. Start the development server:
   ```sh
   bun dev
   ```

## Build

To create a production build:

```sh
bun run build
```
